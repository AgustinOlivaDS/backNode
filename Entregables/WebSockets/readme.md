# Welcome to WebSockets example

Into this repository you'll find a server that have two functionalities, the first one is a html form to create products and use a websocket to update a list of products and at the end, list all products created.
The second functionality is a chat where you can indetify yourself with an email and start to send messages, the page use the websocket to save the messsage in a text file and finally show all the messages with their user and date/time.

## Run server

To run the server in developer mode use the command **npm run dev**, that implements nodemon.
And for production simply use **node index.js**
